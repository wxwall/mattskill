01K3 到底有多强？先看硬指标
先上硬菜。K3 的架构跟之前的所有模型都不一样——它用了月之暗面自研的 Kimi Delta Attention (KDA) 和 Attention Residuals (AttnRes) 架构。别被这些名字吓到，说白了就是：他们重新发明了注意力机制，让模型在大规模参数下还能高效推理。
🧠 MoE 架构：16/896 专家
总共有 896 个专家模块，每次推理只激活 16 个。这意味着虽然模型有 2.8T 参数，但每次实际计算量只相当于一个 300B 左右的密集模型，速度快、成本低。而且用了 Quantile Balancing 技术，专家之间的负载天然均衡，不需要手动调参。
⚡ 量化：MXFP4 权重 + MXFP8 激活
从 SFT 阶段就开始用 4-bit 量化训练，不是训完了再压缩。推理时权重用 MXFP4，激活值用 MXFP8，显存占用大幅降低，在消费级显卡上也能跑。
📈 Scaling 效率：K2 的 2.5 倍
相比上一代 K2，每单位算力带来的性能提升提高了 2.5 倍。这意味着同样的算力投入，K3 能学到更多东西。

02编程能力——跟 Claude 和 GPT 正面硬刚
K3 最让人兴奋的是它的编程能力。月之暗面放了好几个 demo，一个比一个离谱：
🔧 GPU 内核优化
K3 在优化 GPU kernel 上的表现跟 Fable 5 不相上下，大幅超越 Opus 4.8 和 GPT 5.6 Sol。测试覆盖了自研的 KDA、AttnRes 和 MLA 等多个内核，在 NVIDIA H200 和国产 GPU 上都跑了一遍。
🛠️ MiniTriton——从零写 GPU 编译器
K3 从零写了一个精简版 Triton 编译器，包含 tile-level IR、多轮优化 pass 和 PTX 代码生成。最终跑出来的性能跟 Triton 和 torch.compile 相当，甚至在某些场景下更好。这相当于让 AI 自己去写编译器——以前这得是一个编译器团队干几个月的事。
🎮 3D 游戏开发
K3 结合了 3D 推理、编程和视觉能力，通过"视觉闭环"的方式，做出了一个基于 Three.js WebGPU 的 3D 开放世界游戏。代码、画面、交互全部是 AI 自己搞定的。
🔬 芯片设计——48 小时设计一颗芯片
这个是最炸裂的。K3 在 48 小时内自主运行，设计了一颗芯片。在 4mm² 的面积内，塞进了 146 万个标准单元和 0.277MB SRAM，在 100MHz 下时序收敛，仿真解码吞吐量超过 8700 tokens/s。你想想，这活以前一个芯片设计团队得干几个月。
🌌 天体物理研究——2 小时干完 2 周活
K3 用大约 2 小时完成了一个天体物理学家通常需要 1-2 周的工作：复现了 I-Love-Q 通用关系，审阅了 20+ 篇论文，评估了 300+ 个状态方程，生成了 3000+ 行 Python 代码。这不仅仅是写代码——它需要理解物理概念、阅读论文、提取关键信息、然后实现计算。

04Kimi Code + Kimi Work——产品矩阵全面升级
K3 不只是个 API，月之暗面围绕它搭了一整套产品矩阵：
💻 Kimi Code — AI 编程助手
面向开发者的 AI 编程 Agent 和 CLI 工具。支持终端、IDE 和 CLI 三种使用方式，覆盖软件开发生命周期的全部环节。K3 模型在 Moderato 及以上档位可用。安装只要一行命令：
curl -fsSL https://code.kimi.com/kimi-code/install.sh | bash
🖥️ Kimi Work — 桌面 AI Agent
面向知识工作者的桌面 AI 应用。支持本地文件挂载、WebBridge 浏览器自动化（AI 像人一样浏览网页）、24/7 定时任务（内置 Cron 引擎）、Agent Swarm 群体智能、金融数据集成。K3 上线后新增了 Widget 组件和 Dashboard 面板功能。
05K3 的短板——老实说，不完美
有一说一，K3 也不是没有缺点。月之暗面自己承认了三个主要问题：
⚠️
对思考历史敏感
K3 是在"保留思考历史"模式下训练的。如果切换模型或者没有正确传递历史思考内容，质量会明显下降。
⚠️
过度主动
因为太擅长长任务了，遇到小问题或者意图不明确的时候，K3 可能会自作主张做一些你没想到的事。官方建议在 system prompt 里明确约束行为边界。
⚠️
用户体验有差距
官方承认 K3 在整体用户体验上跟 Claude Fable 5 和 GPT 5.6 Sol 还有差距。模型能力很强，但产品化程度还需要时间打磨。

06我的看法——为什么 K3 是个里程碑
说实话，K3 的真正意义不在于它有多少参数，或者在某些 benchmark 上赢了多少分。它的意义在于三件事：
🔥
中国团队做出了世界级的模型
这已经不是"追赶"了。K3 在编程、Agent、知识工作多个维度上跟 OpenAI 和 Anthropic 的旗舰模型正面竞争，有些地方甚至领先。海外社区的震惊反应说明了一切。
🌍
真的开源，不是"开源"
模型权重 7 月 27 日开放，不是那种"开源"但只给个 API 调用的伪开源。这意味着全球的开发者、研究者都能基于 K3 做二次开发。
🏗️
自研架构，不是套壳
KDA 注意力机制、Quantile Balancing、Per-Head Muon 优化器——这些都是月之暗面自己的创新。K3 跟 K2 完全不同，是一次彻底的架构升级。
