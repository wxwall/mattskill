# Mission: ECC — AI 编程助手的"操作系统"

## Why
ECC 是 GitHub 上最火的 AI Agent 配置系统（224K+ Stars），但中文社区对它几乎一无所知。制作一套面向中文开发者的公众号课程，让国内开发者理解 ECC 的价值、能上手使用，并从中受益。

## Success looks like
- 产出 3-5 篇高质量的公众号文章（golden-style HTML）
- 每篇文章有明确的交付物（读者学完能做什么）
- 文章内容覆盖 ECC 的核心概念、安装配置、实战场景

## Constraints
- 公众号发布，必须遵守 golden-style 规范（暖棕金色系、inline style、无 JS、SVG animate 测验）
- 不能有交互式/AI 助手口吻（公众号单向传播）
- 文章开头必须有社交证明横幅（GitHub 数据）
- 不能有"下一课"导航链接

## Out of scope
- ECC 的 Rust 控制面 (ecc2/) 深入开发
- 非中文内容
- 视频教程制作
