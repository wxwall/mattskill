# 公众号课程发布规范

用户确认的课程发布规则已固化到 SKILL.md：golden-style 视觉（暖棕金色系、inline-style、无 JS）、SVG animate 测验（参考 OpenMontage L1）、不使用 GIF 图片、social proof 横幅、无下一课导航、无交互式口吻。

**Why it matters:** 之前课程中的 GIF （no-mistakes L2 的 demo.gif）在公众号中无法渲染。测验格式也曾反复——用户明确要求 SVG animate `<g><animate>` 模式（OpenMontage 风格），而非 div 静态列表。

**Implications:** 以后所有新课程直接按此规范出 HTML，不需要事后修补格式。
