"安装 codebase-memory-mcp 最快只需一行 curl 命令。因为它是静态二进制，不需要 Node.js、不需要 Python、不需要 Docker——下载即用。"
01安装
多种安装方式。最推荐的是 curl 一键安装（不需要任何运行时）：
🌐 curl 安装（macOS/Linux，推荐）
curl -fsSL https://raw.githubusercontent.com/DeusData/codebase-memory-mcp/main/install.sh | bash
📦 包管理器安装
macOS: brew install codebase-memory-mcp
Windows: scoop install codebase-memory-mcp
npm: npm i -g codebase-memory-mcp
Python: pip install codebase-memory-mcp
📌 区别提醒：CodeGraph 需要已安装 Node.js ≥ 22。codebase-memory-mcp 是静态二进制——curl 安装方式直接下载可执行文件，零依赖。npm/PyPI 安装方式也只是包了一层二进制。
02索引项目
安装后，进入项目目录运行索引命令：
cd your-project
codebase-memory index_repository .
这会在内存中构建项目知识图谱。索引速度极快——大多数项目在毫秒级完成。索引完成后，可以用以下命令验证：
codebase-memory index_status
也可以启动可视化 UI 查看图结构：
codebase-memory serve --ui
然后访问 http://localhost:9749 查看可视化图谱。
03在 Claude Code 中配置 MCP
在 claude.json 中添加 MCP Server 配置：
{
  "mcpServers": {
    "codebase-memory": {
      "command": "codebase-memory",
      "args": ["serve", "--mcp"],
      "cwd": "/path/to/your/project"
    }
  }
}
配置好重启 Claude Code，14 个 MCP 工具会自动注册。你可以通过 /mcp 命令查看已连接的 Server。
📌 和 CodeGraph 不同的地方：CodeGraph 只有 1 个主工具 + 7 个隐藏工具。codebase-memory 默认暴露全部 14 个工具，包括 Cypher-like 图查询语言（query_graph）和 ADR 管理。
