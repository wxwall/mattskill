# 初始技术背景

用户已完成 CodeGraph 课程（4 课），对代码知识图谱 + MCP 集成有较深理解。CodeGraph 和 codebase-memory-mcp 解决类似问题，但后者在性能和语言覆盖上更进一步。

**会用到的前置知识：**
- MCP 协议概念（Model Context Protocol）
- tree-sitter AST 解析基本原理
- 知识图谱节点和边
- Claude Code MCP 配置方式

**教学策略：**
- 善用对比：突出 codebase-memory-mcp 与 CodeGraph 的差异
- 重点讲独特卖点：158 种语言、单个二进制、RAM-first 流水线、14 个工具
- 安装和 MCP 配置部分可以加快，用户已经熟悉流程
