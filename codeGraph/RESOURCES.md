# CodeGraph Resources

## Knowledge

- [GitHub: colbymchenry/codegraph](https://github.com/colbymchenry/codegraph)
  官方仓库，包含 README、CHANGELOG、源码。Use for: 安装指南、API 参考、版本变更跟踪。

- [npm: @colbymchenry/codegraph](https://www.npmjs.com/package/@colbymchenry/codegraph)
  npm 包页面。Use for: 版本历史、依赖信息。

- [MCP (Model Context Protocol) 官方文档](https://modelcontextprotocol.io/)
  MCP 协议规范。Use for: 理解 CodeGraph 如何通过 MCP 与 AI Agent 通信。

- [tree-sitter 官方文档](https://tree-sitter.github.io/tree-sitter/)
  CodeGraph 底层使用的语法解析器。Use for: 理解 AST 解析原理、支持的语言。

- [SQLite FTS5 文档](https://www.sqlite.org/fts5.html)
  CodeGraph 使用的全文搜索引擎。Use for: 理解语义搜索实现。
