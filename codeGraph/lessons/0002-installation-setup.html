01安装 CodeGraph
三种安装方式，选一种即可：
🎯 方式一：npx 直接运行（推荐）
无需安装，直接运行。适合首次体验：
npx @colbymchenry/codegraph
📦 方式二：npm 全局安装
常用方式，一次安装全局可用：
npm i -g @colbymchenry/codegraph
🌐 方式三：curl 安装（无需 Node.js）
CodeGraph 自带运行时，无需预装 Node.js：
curl -fsSL https://codegraph.run/install.sh | sh
📌 安装后验证：运行 codegraph --version 查看版本号。如果看到 v1.x.x，说明安装成功。
02初始化项目
进入你的项目目录，运行初始化命令：
cd your-project
codegraph init
codegraph init 会在项目根目录创建 .codegraph/ 目录，包含：
📁
.codegraph/codegraph.db — SQLite 数据库，存储所有符号和关系
📁
.codegraph/codegraph.toml — 配置文件，可自定义索引行为
03运行索引
初始化后，运行索引命令构建知识图谱：
codegraph index
CodeGraph 会自动扫描项目中的源码文件（支持 20+ 种语言），用 tree-sitter 解析 AST，提取符号和关系，写入 SQLite 数据库。一个中型项目只需 1-3 分钟。
📌 索引了什么？每次运行索引都会增量更新——只重新解析修改过的文件，不会全量重扫。首次索引是全量，后续都是增量。
04检查状态和常用命令
codegraph status
查看索引状态：已索引文件数、符号数、最后更新时间
codegraph search <query>
全文搜索代码库中的符号和代码片段
codegraph serve --mcp
启动 MCP Server，等待 AI Agent 连接（第 3 课详解）
codegraph watch
启动文件监听模式，文件变化自动增量索引
