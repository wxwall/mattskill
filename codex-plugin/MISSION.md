# Mission: OpenAI Codex Plugin for Claude Code

## Why
这是一个让 Claude Code 和 OpenAI Codex 联手的插件。学习它能让你理解：AI 编程工具之间的协作模式、插件化的工作流、以及如何用不同模型各取所长。

## Success looks like
- 能说清楚 Codex Plugin 是干什么的、解决什么问题
- 会安装并在 Claude Code 中使用 `/codex:` 系列命令
- 知道 review、rescue、transfer 等核心命令的适用场景
- 理解 review gate 机制和异步任务模式

## Constraints
- 公众号文章形式发布，面向中文开发者
- 保持口语化风格，不能有 AI 味
- 封面图手动生成

## Out of scope
- 深入 Codex CLI 内部实现
- 对比 Claude Code 和 Codex 的能力差异
- OpenAI API 的计费细节
