# Mission: Apple Container

## Why
Apple 开源了一个叫 `container` 的工具，可以在 Mac 上用轻量虚拟机跑 Linux 容器。这是 Apple 官方出的容器方案，Swift 编写，专为 Apple Silicon 优化。我的电脑是 M 系列芯片，想了解这个工具是什么、跟 Docker 有什么区别、能不能在我的场景中用起来。

## Success looks like
- 能用自己的话解释清楚 Apple Container 是什么、怎么工作的
- 掌握基本的安装和使用命令（pull、run、exec、build）
- 知道 Apple Container 和 Docker 的核心区别在哪里
- 能用它跑一个实际的 Linux 容器并执行命令

## Constraints
- 学习时间：业余时间，每次学一个短课程即可
- 语言偏好：中文教学，技术术语保留英文
- 目前没有 macOS 26 环境，主要是知识性学习

## Out of scope
- 不深入 Swift 源码开发
- 不涉及 Containerization Swift Package 的 API 编程
- 不涉及 Podman Desktop 扩展等第三方集成工具
