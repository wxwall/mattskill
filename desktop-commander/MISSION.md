# Mission: Desktop Commander MCP

## Why
了解这个 8.2K+ Stars 的开源 MCP 项目，掌握如何让 AI 助手获得终端控制、文件操作和进程管理能力。

## Success looks like
- 理解 Desktop Commander 作为 MCP 服务器的核心定位
- 知道三种安装方式（npx / Docker / 手动配置）
- 理解安全机制和风险（目录白名单、Docker 隔离、审计日志）
- 知道它支持哪些客户端和真实使用场景

## Constraints
- 需要 Node.js 环境（npx 方式）或 Docker（Docker 方式）
- 安全限制可通过命令绕过，生产环境建议用 Docker

## Out of scope
- 不涉及 MCP 协议本身的深入原理
- 不涉及 Desktop Commander App 的详细使用
- 不涉及具体的安全加固方案
