CodeGraph
⏱ 预计 15 分钟  ·  ← 第 2 课

02安装 CLI
📌 推荐：一键安装脚本
curl -fsSL https://raw.githubusercontent.com/colbymchenry/codegraph/main/install.sh | sh
macOS / Linux。Windows 用 PowerShell 版本
npm 全局安装（需 Node.js）
npm i -g @colbymchenry/codegraph
✅ 验证：codegraph --version→ v1.1.0+
💡 升级：codegraph upgrade自动检测并原地更新
03接入 AI Agent
把 CodeGraph 的 MCP Server 配置写入 Agent 的配置文件。这一步不会索引任何代码——只是告诉 Agent「有 CodeGraph 这个工具可以用」。
codegraph install
①
🔍 自动检测— 扫描机器上安装了哪些 Agent，自动写入对应 MCP 配置
②
📝 写入配置— MCP config 中添加 CodeGraph Server 条目
③
📋 注入指令— CLAUDE.md / AGENTS.md 插入 marker-fenced 指令
📄 MCP 配置示例
{
  "mcpServers": {
    "codegraph": {
      "type": "stdio",
      "command": "codegraph",
      "args": ["serve", "--mcp"]
    }
  }
}

05文件结构与日常使用
📁 生成的文件
your-project/
├──.codegraph/
│ ├──codegraph.db   ← SQLite 数据库
│ └──codegraph.json  ← 项目配置（可选）
└──...
📌 建议：把.codegraph/加入.gitignore
索引文件不提交到 Git——它是派生数据，团队成员各自运行codegraph init即可。
🔄
自动同步
文件变化 → debounce 2s → 增量更新。写完保存，图就更新。
🧹
卸载
codegraph uninstall移除Agent配置 ·codegraph uninit删除索引
🏗️
Monorepo
根目录 init，所有子项目一起索引。Agent 通过 projectPath 目标。
