## 1. 你需要准备什么
CodeGraph 自带运行时，不需要安装 Node.js。下载解压就能用——不需要编译、不需要原生构建、所有平台行为一致。
🖥️ target_os
macOS (x64, arm64)
Linux (x64, arm64)
Windows (x64, arm64)
🤖 supported_agents
Claude Code · Cursor
Codex CLI · Gemini CLI
Antigravity · Kiro
📦 project_reqs
Any language
24 languages
17 frameworks
## 2. 安装 CLI // 三种方式，任选一种
// ✅ 推荐：一键安装脚本
$ curl -fsSL https://raw.githubusercontent.com/colbymchenry/codegraph/main/install.sh | sh
# macOS / Linux · Windows 请用 PowerShell 版本
// npm 全局安装 (requires: node.js)
$ npm i -g @colbymchenry/codegraph
// 验证安装
$ codegraph --version
→ v1.1.0 或更高
$ codegraph upgrade
# 自动检测安装方式（脚本/npm/npx）并原地更新
## 3. 接入 AI Agent
把 CodeGraph 的 MCP Server 配置写入 Agent 的配置文件
$ codegraph install
1. 自动检测 — 扫描机器上安装的 Agent
2. 写入配置 — MCP config 中添加 CodeGraph Server
3. 注入指令 — CLAUDE.md / AGENTS.md 插入 marker-fenced 指令
📝 ~/.claude/.mcp.json
{
  "mcpServers": {
    "codegraph": {
      "type": "stdio",
      "command": "codegraph",
      "args": ["serve", "--mcp"]
    }
  }
}
# 自动设置 mcp__codegraph__* 通配符权限，调用时不需要确认
## 4. 初始化项目
$ cd your-project
$ codegraph init
🔍 Scan 扫描项目文件 → 跳过 node_modules
🌳 Parse tree-sitter 解析 → 提取符号 (24 langs)
🔗 Resolve 解析符号引用关系 → 跨文件 + 框架路由
💾 Store 写入 .codegraph/codegraph.db (SQLite + FTS5)
✅ Done 文件监听器已启动，后续自动同步
## 5. 生成的文件
$ tree your-project/
├── .codegraph/
│ ├── codegraph.db   ← SQLite 数据库（图数据）
│ └── codegraph.json  ← 项目级配置（可选）
└── ...
⚠ WARNING: 建议将 .codegraph/ 加入 .gitignore —— 索引文件不提交到 Git
## 6. 自动同步与日常使用
🔄 auto_sync()
文件监听器检测变化 → debounce 2s → 增量更新索引。写完代码保存，图就更新。
🧹 uninstall()
$ codegraph uninstall # 移除 Agent 配置
$ codegraph uninit    # 删除项目索引
🏗️ monorepo()
根目录 codegraph init → 所有子项目一起索引 → Agent 通过 projectPath 指定目标
