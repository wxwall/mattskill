# 公众号封面图工作流

## 日期
2026-07-06

## 学到了什么
创建课程 HTML 时，需要额外生成一张 2.35:1 的 JPEG 封面图供公众号上传。标准流程：

1. **设计 SVG 封面** — 手绘 1175x500 的 SVG，包含版本标签、主标题、副标题、一句话说明、统计卡片、社区热评
2. **Edge headless 截图** — 用 Microsoft Edge 无头模式渲染 SVG 并截取 PNG
3. **PIL 转 JPEG** — 用 Python 去掉透明通道保存为 JPG
4. **文章结构**：笔记本头部（保留标题）→ 封面图 → 正文，封面图已包含社交证明信息，正文中不再重复

关键教训：不能直接使用 SVG 作为公众号图片；封面图包含社交证明横幅后，正文中不再重复生成。

## 为什么重要
之前封面图和正文头部信息（社交证明横幅）重复，视觉上显得啰嗦。封面图本身已经浓缩了版本、统计数据和社区热评，正文再放同样的内容就是冗余。应当把封面图理解为"视觉化的文章头部"，正文直接从此进入。

## 相关知识
- golden-style：暖棕金色系、inline-style、无 JS、无 GIF
- 封面图 SVG 与 golden-style 配色保持一致（#f5f0e8 背景、#8b6914 金色统计数字等）

## 何时复习
下次创建新课程时参照 SKILL.md 中的"封面图流程"和"文章开头结构"两条规则。
