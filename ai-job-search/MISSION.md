# Mission: AI Job Search — 基于 Claude Code 的 AI 求职框架

## Why
了解这个 22.1K+ Stars 的开源 AI 求职工具，掌握它的核心工作流和双 Agent 简历机制，学会用它来提升求职效率。

## Success looks like
- 理解 AI Job Search 的 5 步核心工作流（/setup → /scrape → /apply → /interview → /outcome）
- 理解"写手-评审"双 Agent 简历机制
- 能独立安装和配置基础环境
- 能运行 /setup 建立个人求职档案

## Constraints
- 工具依赖 Claude Code CLI、Python 3.10+、Bun、LaTeX
- 主要面向海外求职市场（内置 LinkedIn、丹麦求职网站搜索）
- 需通过 /add-portal 添加国内求职网站

## Out of scope
- 不涉及具体求职策略和面试技巧
- 不涉及 LaTeX 模板设计
- 不涉及 Claude Code 本身的安装和使用
