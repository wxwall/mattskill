01一句话说清楚这工具干啥的
AI Job Search 是一个基于 Claude Code 的 AI 求职框架。你把它装到 Claude Code 里，然后通过几个斜杠命令，就能完成搜职位、评估匹配度、写简历、做 cover letter、模拟面试这一整套流程。TypeScript + Python 写的，MIT 协议，完全开源免费。
说白了就是：你告诉 Claude 你想找什么工作，它帮你把剩下的全干了。每份简历都是针对岗位量体裁衣，每封 cover letter 都有理有据。而且它不是那种"一键群发"的垃圾工具——它有一整套质量把控机制，保证每份投递都有水平。

02核心工作流：5 个命令搞定求职
整个工具的核心就是这条流水线：
/setup → /scrape → /apply <url> → /interview → /outcome
1
/setup — 建你的求职档案
把你的简历、LinkedIn 资料、作品集扔进去，Claude 会跟你聊一轮，把你的技能、经验、求职偏好全部结构化存下来。三种方式：导入文件夹、粘贴 CV、或者让 Claude 直接采访你。
2
/scrape — 搜刮职位信息
自动扫描多个求职网站，按匹配度排序。内置了 LinkedIn、Jobindex、Jobbank 等多个搜索工具。结果会按你的技能匹配度打分，一目了然。
3
/apply <url> — 核心：AI 帮你写简历
这是最牛的部分。把职位链接丢进去，Claude 会做一整套流程：评估匹配度 → 量身定制 CV → 写 cover letter → 生成 PDF → 检查排版 → 模拟 ATS 解析 → 输出完整投递包。后面细说。
4
/interview — 模拟面试
针对你投递的岗位，生成面试准备包 + 模拟面试。Claude 会扮演面试官，按岗位要求问你问题，然后给反馈。
5
/outcome — 记录结果
面试完了记一笔：过了、挂了、到哪一轮了。所有投递记录自动同步到一个 CSV 跟踪表里。
03最牛的部分：/apply 的"写手-评审"双 Agent 机制
这工具最骚的设计就是这个"写手-评审"双 Agent 机制。你想想，你平时写简历是怎么写的？自己写完了自我感觉良好，但根本不知道 recruiter 看了会怎么想。这工具直接解决了这个问题：
✍️
第一步：写手 Agent 初稿
分析岗位描述，跟你的档案做匹配，写一份量身定制的 LaTeX 简历和 cover letter。CV 必须正好 2 页，cover letter 正好 1 页，多一行都不行。
🔍
第二步：评审 Agent 挑刺
第二个 Claude Agent 上场——它假装不认识第一个 Agent，先去调研这家公司，然后回来疯狂挑刺：这里写得太空泛了、那里跟岗位匹配度不够、这个技能应该往前放……
🔄
第三步：迭代修改
根据评审意见修改，改完了再编译成 PDF，检查排版是否正常。如果内容溢出了第 2 页，它会自动裁剪——但不是简单截断，而是按"跟岗位的相关性"打分，删最不相关的行。
🤖
第四步：模拟 ATS 解析
用 pdftotext 提取 PDF 的文本层，模拟 ATS 系统的解析结果。检查联系方式有没有被正确识别、阅读顺序对不对、关键词覆盖率够不够。如果 ATS 解析有问题，继续改。
✅
第五步：输出投递包
最终呈现给你一份完整的投递材料 + 验证清单。你过目一遍，没问题就可以投了。
📌 关于"诚实"这件事
这工具有个底线：绝不造假。简历和 cover letter 里的所有内容，都会被验证是否在你的档案里有据可查。技能不够就是不够，不会硬塞关键词。面试准备也会教你诚实地回答 gap，而不是编经历。有一说一，这比那些"帮你作弊"的求职服务靠谱多了。
04安装上手——半小时搞定
装这玩意儿你需要几样东西：
🤖
Claude Code CLI
核心引擎，所有 Agent 操作跑在它上面
🐍
Python 3.10+
跑一些辅助脚本
🍞
Bun
JavaScript 运行时，用来跑各个求职网站的 CLI 搜索工具

📌 快速安装
# 1. Fork 并克隆
gh repo fork MadsLorentzen/ai-job-search --clone
cd ai-job-search

# 2. 安装各求职网站的 CLI 工具
for tool in jobbank-search jobdanmark-search jobindex-search jobnet-search linkedin-search freehire-search; do
cd .agents/skills/$tool/cli && bun install && cd ../../../..
done

# 3. 启动 Claude Code
claude

# 4. 在 Claude Code 里建立档案
/setup
注意：linkedin-search 和 freehire-search 不需要装依赖，直接跑。

💡 给国内用户的建议
这工具目前内置的搜索工具主要是丹麦的求职网站（Jobindex、Jobbank 等）和 LinkedIn。如果你在国内求职，可以用 /add-portal 添加国内的招聘网站。不过简历生成和 cover letter 功能是通用的，不管你在哪个国家都能用。
06几个你可能不知道的细节
📐
CV 必须正好 2 页，多一行都不行
系统编译 PDF 后会检查页数，如果超了会自动裁剪。裁剪不是简单截断——它会按"跟这个岗位的相关性"给每一行内容打分，删最不相关的。而且还会考虑 cover letter 是否依赖了这行内容，如果依赖了就不删。
🔍
ATS 解析检查
很多大公司第一轮筛简历是 ATS 系统自动解析。如果 PDF 的文本层乱码，或者阅读顺序不对，系统直接拒。这个工具会用 pdftotext 模拟 ATS 解析，确保你的简历能被机器正确读取。
🎯
职业探索模式
如果你不确定自己想做什么方向，这工具会分析你的完整经历——包括哪些工作让你有干劲、哪些让你想死——然后帮你发现一些你自己都没意识到的职业方向。
💰
薪资对标工具
内置了一个 salary_lookup.py 脚本，可以帮你做薪资对标。你只需要提供一些薪资数据（工会统计、Glassdoor、自己做调研），就能算出合理的期望薪资范围。
