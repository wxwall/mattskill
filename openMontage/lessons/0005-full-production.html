01全栈模式的 API 全家桶
全栈模式需要配置多个 API Key，每个解锁一层能力：
🎥
视频生成 API
Google Veo 3、Kling、Runway Gen-4、MiniMax——从文字/图片生成运动视频片段。这是从"图片幻灯片"到"真视频"的关键一步。
🗣️
高级 TTS
ElevenLabs——媲美真人的语音合成，支持多语言、情感表达。也可选 Google TTS、OpenAI TTS。
🎵
AI 音乐
Suno——AI 生成原创配乐。也可以接入 ElevenLabs 音乐生成。
🧑
数字人/Avatar
HeyGen——AI 发言人视频，数字人 + TTS + 图文叠加。
📌 不用全配：按需选择。想做电影预告片 → Veo + ElevenLabs + Suno。想做 AI 发言人 → HeyGen + ElevenLabs。想省钱 → 只加视频 API，TTS 继续用 Piper。
02配置 .env
在 .env 文件中添加对应的 Key：
# 视频生成（选一个或多个）
GOOGLE_API_KEY=your_google_key
RUNWAY_API_KEY=your_runway_key

# 高级 TTS
ELEVENLABS_API_KEY=your_elevenlabs_key

# AI 音乐
SUNO_API_KEY=your_suno_key
Agent 会自动检测哪些 Key 已配置，选择最佳工具组合。如果某个 Key 没配，Agent 会自动降级到免费替代方案。

04三种模式全景对比
🆓 零密钥

$0 · 5-15 分钟 · 适合学习验证和简单科普

所有零密钥能力 + FLUX 定制图片 + 动画 + 粒子
$0.15–1.00 · 10-20 分钟 · 适合自媒体和产品视频

所有能力 + 视频生成 + 高级 TTS + AI 音乐 + 数字人
$1–3 · 15-40 分钟 · 适合商业项目和广播级内容
