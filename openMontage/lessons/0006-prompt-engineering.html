01好 Prompt 的五个要素
1
🎯 指定视频类型和时长
60-second explainer？30-second trailer？不同类型的视频结构和节奏完全不同。45s≈110词，90s≈225词。
2
👥 明确目标受众
for middle school students → 简单语言、活泼视觉。for developers → 技术术语、代码展示。
3
📊 列出想要的视觉组件
bar charts、pie charts、comparison cards、callout boxes、KPI grids——Agent 支持这些图表类型。
4
🎨 描述风格和氛围
Ghibli-style、cinematic、clean professional、dark moody——风格关键词决定视觉方向。
5
💰 指定预算约束（可选）
"Use only free tools" → 零密钥。什么都不说 → Agent 用最优工具（可能花钱）。
02同一主题，好 Prompt vs 差 Prompt
❌ 差 Prompt
"Make a video about climate change."
→ Agent 不知道：多长？什么类型？给谁看？什么风格？结果往往是泛泛而谈的幻灯片。
✅ 好 Prompt
"Create a 90-second animated explainer about climate change for high school students. Include temperature rise line charts, CO2 emission bar charts, before/after comparison cards, and a hopeful call-to-action closing. Use only free tools."
→ Agent 知道：时长、受众、图表类型、结尾风格、预算。产出精准匹配需求。
03按场景的 Prompt 模板
🎓 教育/科普
"Create a [时长]-second animated explainer about [主题] for [受众]. Include [图表类型]. Use [预算模式] tools."
🚀 产品发布
"Make a [时长]-second product launch video for [产品名]. Highlight [3 个功能] with metric cards and comparison views. Target [受众]."
🎬 电影/创意
"Create a [时长]-second [风格] video of [场景描述]. Describe atmosphere, lighting, and mood in detail. Use [预算]."
📱 社交媒体
"Take my [内容/URL] and turn it into a [时长]-second video for [平台]. Use word-by-word captions. [风格] pacing."
04常见错误和纠正
⚠️
太简短 → 结果泛泛
"Make a video about AI" → Agent 无从下手。加上时长、受众、组件。
⚠️
不指定图表类型 → 只有文字
Agent 不会自动猜到你要柱状图。说出：bar chart、pie chart、KPI grid。
⚠️
不指定预算 → 可能花钱
Agent 默认选最优方案。想免费必须说 "Use only free tools"。
⚠️
风格描述太模糊 → 视觉不统一
不是"looks good"，而是"Ghibli-style with warm golden lighting and floating particle effects"。
