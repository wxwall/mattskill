01一句话理解 OpenMontage
OpenMontage 是一个开源的、Agent 驱动的视频制作系统。它不是一个独立的应用——它寄生在 AI 编程助手（Claude Code、Cursor、Copilot 等）里，把 AI Agent 变成一个视频导演。

02没有中心调度器——Agent 就是导演
OpenMontage 最核心的设计理念：没有中心调度器。不像传统视频软件有一个固定的 UI 流程，OpenMontage 让 AI Agent 自己做导演。Agent 读取 YAML 流水线定义和 Markdown 技能文件，自主决定每一步怎么做。
第 1 层：tools/ + pipeline_defs/
「有什么」—— 52 个 Python 工具 + 12 个 YAML 流水线定义。这是可执行的能力层。
第 2 层：skills/
「怎么用」—— 400+ Markdown 技能文件，教 Agent 每个制作阶段的质量标准和操作规范。
第 3 层：.agents/skills/
「为什么这样工作」—— 外部技术知识包（API 文档、格式规范等）。

04免费也能跑——成本模式
🆓 零密钥模式 — 完全免费
Piper TTS（离线语音）+ Pexels/Pixabay（免费素材）+ Remotion（React 合成）→ 可以做出带旁白、图表、字幕的真视频。有预置 Demo 可以直接跑：make demo
🔑 单密钥模式 — ~$0.50–$1.50
加一个 FAL_KEY 解锁 FLUX 图片生成，AI 自动生成配图 + Ken Burns 动画 + 字幕。30 秒动漫风格视频最低 $0.15。
🎬 全栈模式 — ~$1–$3
视频生成 API（Veo/Kling/Runway）+ ElevenLabs 高质量 TTS + Suno 音乐 → 广播级成品。
