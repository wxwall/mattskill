01零密钥技术栈
不花一分钱的三件套：
🗣️
Piper TTS — 离线语音合成
完全在本地运行，不需要联网。支持多种语言和声音。质量不如 ElevenLabs，但免费且无限使用。
📦
免费素材库
Pexels / Pixabay（免费图片视频，需注册免费 API Key）· Unsplash（免费图片）· Archive.org / NASA / Wikimedia Commons（公共领域素材）
🎬
Remotion — React 视频合成引擎
用 React 组件写视频——文字动画、图表、转场、字幕。最终通过 FFmpeg 渲染为 mp4。
02先跑预置 Demo
OpenMontage 内置了 3 个零成本的预渲染 Demo，不需要 Agent 参与：
# 列出所有 Demo
./render-demo.sh --list

# 运行某个 Demo
./render-demo.sh world-in-numbers
./render-demo.sh code-to-screen
./render-demo.sh focusflow-pitch

# 或一次性全部
make demo
🌍 world-in-numbers
45s · KPI 网格、柱状图、饼图、折线图、统计卡片
$0
💻 code-to-screen
50s · HTTP 生命周期、进度条、标注框
$0
🚀 focusflow-pitch
40s · 创业 pitch、增长指标、收入环形图、客户见证
$0
📌 建议：先跑通 world-in-numbers。这是最快的验证方式——如果它渲染成功，说明你的 Python + FFmpeg + Node.js + Remotion 环境全部正常。

04更多零密钥 Prompt 示例
这些 Prompt 在零密钥模式下都已验证可用（~8-15 分钟，$0）：
☕ 数据可视化
"Create a 60-second data-driven video about coffee consumption around the world. Use only free tools."
→ 柱状图（国家对比）、饼图（咖啡类型）、旁白解说
🌐 技术概念
"Make a short explainer about how the internet works. Use only free tools."
→ 章节标题、文字卡片、统计数据、逐词字幕
🔀 开发者内容
"Create a 90-second animated explainer about how Git rebase works. Use only free tools."
→ Rebase vs Merge 对比卡片、提示框、分步动画
05理解输出文件
渲染完成后，在项目目录下会生成：
output/
├── sky-blue-explainer.mp4   ← 最终视频
├── narration.wav          ← 旁白音频
├── subtitles.srt        ← 字幕文件
├── script.md            ← 脚本文件
└── research.md          ← 研究笔记
每个文件都可以单独使用。脚本不满意？改 script.md再重新渲染。字幕想调整？改 subtitles.srt。
