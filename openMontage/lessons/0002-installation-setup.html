📚 课程笔记 · 0002
安装与环境配置
让 OpenMontage 跑起来
⏱ 预计 15 分钟 · ← 第 1 课
"OpenMontage 依赖三个基础软件：Python、FFmpeg、Node.js。装好它们，再配一个 AI 编程助手，整个系统就跑起来了。API Key 是可选的——不配也能做真视频。"
01四大依赖一览
OpenMontage 需要这四样东西，缺一不可：
1
🐍 Python 3.10+
运行 52 个工具（视频生成、图像、TTS、分析等）。必须是 3.10 或更高版本。
2
🎬 FFmpeg
视频编解码、拼接、后期处理。大部分系统已自带或可通过包管理器安装。
3
📦 Node.js 18+
Remotion（React 合成引擎）和 HyperFrames 的运行环境。
4
🤖 AI 编程助手
Claude Code / Cursor / Copilot / Codex / Windsurf —— 任选一个。
02各平台安装命令
🍎 macOS
Python:brew install python@3.12
FFmpeg:brew install ffmpeg
Node.js:brew install node
🐧 Linux (Ubuntu/Debian)
Python:sudo apt install python3 python3-pip
FFmpeg:sudo apt install ffmpeg
Node.js:curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - && sudo apt install nodejs
🪟 Windows
Python:从 python.org 下载安装包，勾选「Add Python to PATH」
FFmpeg:winget install ffmpeg或从 ffmpeg.org 下载
Node.js:从 nodejs.org 下载 LTS 版本安装包
📌 验证安装：装完后逐条检查——python3 --versionffmpeg -versionnode --version三个都能输出版本号才算就绪。
03克隆仓库 + make setup
三个依赖装好后，克隆 OpenMontage 并运行初始化：
git clone https://github.com/calesthio/OpenMontage.git
cd OpenMontage
make setup
make setup做了这些事：
①创建 Python 虚拟环境（venv）
②安装所有 Python 依赖（requirements.txt）
③安装 Remotion 合成引擎的 Node.js 依赖
④创建 .env 配置文件模板（从 .env.example 复制）
04配置 .env 文件
make setup 会生成 .env文件。打开它，根据需要填入 API Key：
📄 .env 文件结构（按需填写）
# 零成本模式——以下全部留空即可
# 单密钥模式——只需填 FAL_KEY
FAL_KEY=your_fal_key_here
# 全栈模式——按需填入
ELEVENLABS_API_KEY=your_elevenlabs_key
RUNWAY_API_KEY=your_runway_key
GOOGLE_API_KEY=your_google_key
SUNO_API_KEY=your_suno_key
📌 建议：一开始全部留空。先跑通零密钥 Demo，确认系统正常。再逐步添加 API Key 解锁更多能力。
05接入 AI 编程助手
OpenMontage 支持 5 种 AI 编程助手，它们在 OpenMontage 目录下都有对应的配置文件：
🧠
Claude Code
配置文件：CLAUDE.md。OpenMontage 目录下直接运行 claude 即可。
🔲
Cursor
配置文件：CURSOR.md + .cursor/rules/
📋
GitHub Copilot
配置文件：COPILOT.md + .github/copilot-instructions.md
⚡
Codex CLI / Windsurf
配置文件：CODEX.md / .windsurfrules
这些配置文件已经写好了——它们指向共享的 AGENT_GUIDE.md 和 PROJECT_CONTEXT.md，告诉 AI 助手如何调度 OpenMontage 的工具流水线。你不需要改任何东西。
06验证整环境就绪
运行以下命令，三项都成功说明环境就绪：
# 1. Python 环境
python3 -c "import sys; print(sys.version)"
# → 3.10+ 即可

# 2. FFmpeg
ffmpeg -version | head -1
# → 显示版本号

# 3. Node.js + Remotion
cd remotion-composer && npm ls remotion
# → 显示 remotion 版本
✅
三项全部通过 →环境就绪，可以进入第 3 课，跑你的第一个视频了。
