# Mission: Strix

## Why
Strix 是一个开源的 AI 渗透测试工具，能用 AI Agent 自动找漏洞、验证漏洞、甚至修漏洞。GitHub 37K+ Stars，最近火得不行。我想搞清楚这玩意儿到底怎么用、能不能用在我的项目安全测试上。

## Success looks like
- 能用大白话讲清楚 Strix 是什么、怎么工作的
- 能在本地安装并跑一次扫描
- 理解三种扫描模式（quick/standard/deep）的区别
- 知道怎么把 Strix 集成到 CI/CD 流水线
- 理解 AI 渗透测试的局限性和风险

## Constraints
- 学习时间：业余时间，每次学一个短课程
- 语言偏好：中文教学，技术术语保留英文
- 需要 Docker 环境，需要 LLM API Key

## Out of scope
- 不深入 Strix Agent 内部架构
- 不涉及 Strix 云平台的企业功能
- 不涉及实际渗透测试攻击技术（只学工具使用）
