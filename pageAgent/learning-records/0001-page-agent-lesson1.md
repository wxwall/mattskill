# Page Agent — 第 1 课完成

## 2026-07-07

通过 Alibaba Page Agent 开源项目（GitHub 24.8K+ Stars），学习了 GUI Agent 的核心概念。课程已完成第 1 课：《Page Agent 是什么？》并发布为公众号草稿。

## 关键内容
- Page Agent 是一个纯前端 JS 库，通过「读 DOM → LLM 分析 → 操作页面」的流程让网页听懂人话
- 一行 script 标签即可接入，支持 CDN 和 NPM 两种方式
- 不需要浏览器插件、不需要后端、不需要截图——纯文本 DOM 操作
- LLM 兼容性广：通义千问、GPT、Claude、本地模型都支持
- 扩展能力：Chrome 扩展（跨 Tab）+ MCP Server（Beta，外部控制）
