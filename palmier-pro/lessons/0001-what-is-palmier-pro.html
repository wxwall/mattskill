01Palmier Pro 是什么？
Palmier Pro 是一款 macOS 原生 AI 视频编辑器，由 Y Combinator S24 孵化的 Palmier 公司开发。它从零用 Swift 构建，对标 Adobe Premiere Pro 的体验，但将生成式 AI 直接编织到时间线编辑工作流中。
🎬
传统视频编辑器的困境
生成画面（Sora/Runway/Kling）→ 下载 → 导入 → 剪辑，来回切换多个工具，流程割裂
🚀
Palmier Pro 的方案
在时间线内直接调用 AI 模型生成视频/图片/音频，生成即导入，无缝剪辑
📌 开发者怎么说
"This is the video editor AI workflows have been waiting for."
"Finally, a native Mac editor that treats AI as a first-class citizen."
"Premiere-like experience with AI built in, not bolted on."
— GitHub Issues 和社区反馈
02核心功能速览
✂️
多轨道时间线编辑
视频、音频、图片、文字轨道，支持裁剪、分割、变速、透明度、变换
🤖
内置 AI 生成
Seedance、Kling V3、Veo 3.1、Grok Imagine 等模型直出到时间线
🔌
MCP Agent 集成
暴露 MCP 服务器，Claude/Cursor/Codex 可直接操作你的项目
📤
多种导出方式
MP4（H.264/H.265）、ProRes，以及 NLE XML（导入 Premiere/DaVinci）
03系统要求与安装
Palmier Pro 目前仅限 macOS 平台：
💻
macOS 26 (Tahoe)
最低操作系统版本要求
🔲
Apple Silicon 专属
仅支持 M 系列芯片，不支持 Intel Mac
📦
DMG 安装包
从 GitHub Releases 下载，拖拽安装
📌 安装步骤
1. 打开 GitHub Releases 页面：github.com/palmier-io/palmier-pro/releases
2. 下载最新版的 .dmg 文件
3. 打开 DMG，将 Palmier Pro 拖入 Applications 文件夹
4. 首次启动可能需要允许未验证的开发者应用
