01什么是 MCP？
MCP（Model Context Protocol）是由 Anthropic 提出的开放协议，规范 AI 应用如何与外部工具交互。可以理解为"AI 世界的 USB 接口"。
🔌
传统方式
每个 AI 工具各自写一套 API 集成。工具 A 接 Claude，工具 B 也要重写一遍。
✅
MCP 方式
一次实现 MCP 服务器，对接所有 MCP 客户端（Claude Code、Cursor、Codex、Claude Desktop 等）。
02Palmier Pro 的 MCP 服务器
当 Palmier Pro 运行时，后台自动启动 MCP 服务器：
http://127.0.0.1:19789/mcp
📂
读取项目信息
时间线结构、轨道内容、片段属性
✏️
编辑时间线
添加/删除片段、调整顺序、修改属性
🤖
调用 AI 生成
Agent 自主调用模型生成并插入时间线
📤
导出操作
Agent 完成编辑后触发导出
03连接你的 AI Agent
🤖 Claude Code
claude mcp add --transport http palmier-pro http://127.0.0.1:19789/mcp
🔧 Codex
codex mcp add palmier-pro --url http://127.0.0.1:19789/mcp
🖱️ Cursor
通过 Help → MCP Instructions → Install in Cursor 一键安装，或手动编辑 ~/.cursor/mcp.json
💬 Claude Desktop
Help → MCP Instructions → Install in Claude Desktop 一键安装（mcpb 扩展包）。安装后直接在对话中编辑视频。
04实战对话示例
💬 和 Claude Code 对话编辑视频
你："帮我看看当前项目的时间线结构"
Claude："当前 3 个轨道：视频轨道（2 片段）、音频（1 片段）、文字（1 标题）"
你："在末尾添加一段 Kling V3 生成的城市夜景延时，赛博朋克风格"
Claude："已调用 Kling V3 生成，自动插入时间线末尾"
📌 典型应用场景
• 批量编辑："把所有片段速度改为 1.5x"
• AI 填充："在 3 个空位各生成 5 秒素材，统一风格"
• 自动字幕："为音频轨道生成字幕"
• 一键成片："根据这些素材自动剪辑成 30 秒短视频"
• 审核修改："把第三段换成图片，加淡入效果"
05Agent-in-the-Loop 协作
Palmier Pro 的设计理念是 Agent-in-the-Loop：你在环中把控创意方向，Agent 负责执行。
👤
你：定方向、审片、决策
把控创意方向和最终质量，在时间线中手动精调
🤖
AI Agent：执行者
快速完成重复操作、批量处理、AI 素材生成，通过 MCP 读写时间线
