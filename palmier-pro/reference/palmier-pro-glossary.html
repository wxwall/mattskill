Palmier Pro 术语表
核心概念
Palmier Pro
开源的 macOS 原生 AI 视频编辑器。Y Combinator S24 孵化。
时间线 (Timeline)
多轨道编辑界面，支持视频、音频、图片、文字轨道叠加剪辑。
MCP (Model Context Protocol)
模型上下文协议。Palmier Pro 通过 MCP 暴露接口，AI Agent 可直接操作时间线。
AI 模型
Seedance 2.0
字节跳动 AI 视频生成模型。
Kling V3
快手科技 AI 视频生成模型。
Veo 3.1
Google AI 视频生成模型。
Grok Imagine
xAI 图像生成模型。
技术概念
macOS 26 (Tahoe)
2026 年 macOS 版本。Palmier Pro 最低要求。
Apple Silicon
苹果 M 系列芯片，不支持 Intel Mac。
NLE XML
非线编 XML，可导入 Premiere Pro / DaVinci Resolve。
ProRes
苹果专业视频编码格式。
